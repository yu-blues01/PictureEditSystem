OpenCVを用いて作った画像加工アプリケーションです
入力された画像内の人物の顔(正面のみ)を検出し、絵文字(☺)で隠し、加工を示すウォーターマーク画像を画像右下部に合成します。
合成後画像をブログサイズとサムネイルサイズにリサイズし、それぞれを保存します。

アプリケーションの処理の流れ
1．アプリケーションを起動する
 　PictureEditSystemフォルダ ⇒ distフォルダ ⇒ PictureEdit.exeを実行
2．「画像フォルダーを選択してください」というポップアップが出るため、加工を施したい画像ファイルがあるディレクトリを選択する
3．選択したディレクトリ内に「output\blog」ディレクトリと「output\thumbnail」ディレクトリが自動で作成され、加工済画像が保存される
 　「output\blog」ディレクトリには顔隠し加工とblogサイズにリサイズを施された画像が、「output\thumbnail」ディレクトリには顔隠し加工とblogサイズにリサイズされた画像が、それぞれ保存されます
